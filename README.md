# bsdx

A minimal KOOK bot built with Go and WebSocket.

## Setup

```bash
cp .env.example .env
# Edit .env and fill in your BOT_TOKEN
```

## Run

```bash
go run .
```

## Build

```bash
go build -o bsdx .
```

## Dependencies

- [golang-bot](https://github.com/kaiheila/golang-bot) — KOOK official Go SDK (local replace)
